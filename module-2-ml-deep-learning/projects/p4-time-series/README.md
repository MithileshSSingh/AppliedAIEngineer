# P4: Time Series Forecasting

## Objective
Forecast product demand using time series methods. Compare statistical (ARIMA) and ML-based (Prophet) approaches.

## Dataset
Synthetic daily sales data generated by `src/generate_data.py`:
- 3 years of daily sales for 3 product categories
- Trend, yearly seasonality, weekly patterns, holiday effects
- Some anomalies and missing values

## Key Skills
- Time series decomposition and stationarity testing
- ARIMA model selection (ACF/PACF)
- Prophet with holidays and changepoints
- Forecast evaluation (MAE, RMSE, MAPE)
- Multi-step ahead forecasting

## Suggested Approach
1. Load and explore the time series data
2. Decompose and test for stationarity
3. Fit ARIMA on one product category
4. Fit Prophet with seasonality and holidays
5. Compare forecasts on a held-out test period
6. Forecast all 3 categories and visualize

## How to Run
```bash
python src/generate_data.py
jupyter lab notebook.ipynb
```
