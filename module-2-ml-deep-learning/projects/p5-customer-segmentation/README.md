# P5: Customer Segmentation with Clustering

## Objective
Segment customers using unsupervised learning (K-Means, DBSCAN) and dimensionality reduction (PCA). Create actionable customer profiles.

## Dataset
Synthetic e-commerce customer data generated by `src/generate_data.py`:
- 2,000 customers with RFM metrics and behavioral features
- Hidden segments with distinct purchasing patterns

## Key Skills
- Feature scaling for clustering
- K-Means with elbow and silhouette analysis
- DBSCAN for density-based clustering
- PCA for visualization
- Segment profiling and business interpretation

## Suggested Approach
1. Load and explore customer data
2. Feature engineering (RFM scores)
3. Scale features
4. Determine optimal K (elbow + silhouette)
5. Apply K-Means and DBSCAN
6. Visualize with PCA
7. Profile segments and name them
8. Business recommendations per segment

## How to Run
```bash
python src/generate_data.py
jupyter lab notebook.ipynb
```
