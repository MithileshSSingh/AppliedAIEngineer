# P5: ML ROI & Stakeholder Presentation

## Objective
Build a compelling business case for the churn prediction ML system. Quantify the ROI, create executive dashboards, and present findings to non-technical stakeholders.

## Deliverables
1. **ROI Analysis** — Full financial model with sensitivity analysis
2. **Streamlit Dashboard** — Interactive monitoring dashboard
3. **Executive Summary** — 1-page non-technical summary
4. **Slide Deck Outline** — 5-slide presentation template

## The Story to Tell
```
Problem: $X/month lost to churn
Solution: ML model catches Y% of at-risk customers
Result: $Z/month saved, ROI = N%
Ask: Approve continued investment for Model v2
```

## Key Skills
- Financial modeling for ML
- Streamlit dashboarding
- Data storytelling
- Executive communication

## How to Run
```bash
# ROI analysis
jupyter lab notebook.ipynb

# Interactive dashboard
pip install streamlit plotly
streamlit run src/dashboard.py
```
